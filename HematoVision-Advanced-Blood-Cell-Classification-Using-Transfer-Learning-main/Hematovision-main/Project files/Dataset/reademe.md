This project uses the blood cell Image Classification dataset from Kaggle, which contains images of 4 different types of blood cells.

([https://www.kaggle.com/datasets/paultimothymooney/blood-cells/data](https://www.kaggle.com/datasets/paultimothymooney/blood-cells/data))

To use this dataset: 1.Create a free Kaggle account. 2.Download the dataset. 3.Extract the files and place them in the appropriate folder as per project instructions.
